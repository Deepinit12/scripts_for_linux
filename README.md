Script Collection for Linux / Network Administration

Этот репозиторий содержит коллекцию Bash и Python скриптов для мониторинга, сетевого администрирования и автоматизации.

Скрипты упорядочены по категориям и папкам, чтобы легко ориентироваться и использовать.
_________________________________________________________________________________________________________________________

Monitoring

Скрипты для контроля состояния системы и сервисов:

Bash: cpu_load_tracker.sh, diskfullalert.sh, log_monitor.sh

Python: log_analyzer.py, redis_monitor.py, system_metrics.py

Примеры задач: слежение за нагрузкой CPU/RAM, мониторинг дисков, анализ логов и состояния сервисов.

__________________________________________________________________________________________________________________________

Networking

Скрипты для работы с сетью и удалёнными сервисами:

Bash: iface_recovery.sh, network_diag.sh, port_scanner.sh

Python: api_health_check.py, api_modern_checker.py, arp_table_parser.py, ip_change_notifier.py, port_scanner.py, traceroute.py, website_checker.py

Примеры задач: проверка сетевых интерфейсов, диагностика сети, сканирование портов, анализ ARP-таблиц, мониторинг API и сайтов.

_____________________________________________________________________________________________________________________________

Utils

Полезные утилиты для автоматизации задач:

Bash: backup_rotate.sh, process_tree_killer.sh, service_recovery.sh, ssh_mass_exec.sh

Python: config_validator_yaml.py, ssh_executor.py

Примеры задач: ротация бэкапов, массовое выполнение команд по SSH, проверка конфигураций YAML, управление процессами и сервисами.
